# 🕹️ Escape Game Builder Projet étudiant réalisé en **HTML + CSS + JavaScript**. C’est une petite application web qui permet de **créer et jouer à des escape games interactifs** (façon Genially). --- ## 🚀 Démo en ligne 👉 [Jouer maintenant](https://tonpseudo.github.io/escape-game/) *(remplace `tonpseudo` par ton pseudo GitHub et `escape-game` par le nom de ton dépôt)* --- ## ✨ Fonctionnalités - Créer plusieurs **scènes** avec une image de fond. - Ajouter des **hotspots** (zones cliquables pour naviguer). - Ajouter des **verrous numériques** (codes à 4 chiffres). - Ajouter des **mots de passe** (anagrammes / énigmes textuelles). - Ajouter du **texte** pour donner des indices. - Débloquer des **variables** quand un verrou est résolu (pour conditionner la suite du jeu). - **Exporter** et **importer** ton projet en JSON. --- ## 🛠️ Utilisation 1. Ouvre [l’application en ligne](https://tonpseudo.github.io/escape-game/). 2. Clique sur **Éditeur** : - Ajoute une scène (choisis une image de fond). - Ajoute des éléments (Hotspot, Verrou, Mot de passe, Texte). - Personnalise leurs propriétés. 3. Clique sur **Jouer le projet** pour tester ton escape game. 4. Clique sur **Exporter JSON** pour sauvegarder ton projet. 5. Clique sur **Importer JSON** pour le recharger plus tard. --- ## 📂 Structure
